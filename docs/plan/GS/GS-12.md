# GS-12: UX Polish — Render, Messaging, Docker Build Output

blocked_by: [GS-2, GS-3]
unlocks: []

## Problem
1. Docker image build output floods the terminal — should use single-line scrolling progress
2. Merge outcome renders as `{:?}` debug format — should be human-readable
3. `alpine/git` image assumed always available — should handle missing image
4. Commit attribution uses hardcoded "claude-container" when no git config — should use host user
5. Squash-base staleness is silent — should warn
6. Relative path display doesn't work from all cwds

## Scope
- Docker build: stream layers with single-line overwrite (`\r` trick)
- Human-readable merge outcome display
- Pull/validate alpine/git before first use
- Use host git config for commit attribution
- Warn on stale squash-base during merge
- Test relative path display from various cwds
